☁️ Configurar Sincronización

✅ Conectado. Hogar ID:  ·  Desconectar

Sigue estos pasos una sola vez. Después tú y tu novia solo necesitan abrir la app.

1

Crear proyecto Firebase (gratis)

Abre console.firebase.google.com, pulsa Agregar proyecto, ponle un nombre (ej. inventario-hogar) y sigue los pasos.

2

Crear la base de datos

En el menú izquierdo → Compilación → Realtime Database → Crear base de datos. Elige la región más cercana (ej. us-central1) y selecciona modo de prueba.

3

Obtener la configuración

Ve a Configuración del proyecto (icono ⚙️) → Tus apps → </> Web → Regístrala. Copia el objeto firebaseConfig y pégalo abajo.

4

Elegir un código de hogar

Invéntate un código secreto compartido con tu novia (ej. casabenja2025). Ambos deben usar el mismo exacto.
